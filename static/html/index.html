Enter NightBot API Credentials

Instructions

  1. Log in to the NightBot Applications Page
  2. Click the "New App" button.
  3. Enter Nightbot Go for the name, and http://localhost:5775 in the "Redirect URIs" box. Then click "Submit".
    • the Redirect URI value you use may be different if you are running NightBot Go on a server or on a different port
    • the name you use for the application is not important - you can use any value
  4. Now click the pencil icon next to "NightBot Go" in the My Apps section, and click "New Secret" in the pop up box.
  5. Copy and paste the Client ID and Client Secret in the form above.
  6. Finally, hit the submit button on this page. You will be redirected to NightBot to authorize this application to access some of your NightBot data, such as your queue information and channel information.
